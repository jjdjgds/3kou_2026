# bowlingGame

## 概要
C++とDirectX 11を使用した3Dボウリングゲームです。

## 開発環境
- **言語**: C / C++
- **API**: DirectX 11
- **IDE**: Visual Studio 2022
- **バージョン管理**: Git / GitHub






## アピールポイント

### 1. 演出を強化するカメラワークとエフェクト
プレイヤーがピンに近づく際の臨場感を高めるため、ボールの動きに連動したカメラワークを実装しました。また、ピンが倒れた際の演出に注力し、視覚的な爽快感を生み出すエフェクト処理を構築しています。

### 2. オブジェクト指向による拡張性の高い設計
DirectX 11を用いた低レイヤーな実装においても、コードの保守性を確保するため、クラス構造を明確に定義しました。将来的な機能追加やルールの変更に柔軟に対応できる設計を意識しています。

## 操作方法
- **パワー決定**: Pキー
- **ボール発射**: Oキー
